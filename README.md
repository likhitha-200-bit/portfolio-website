#  Likhitha-200-bit – Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio highlights my skills, projects, experience, and contact information with a clean and interactive user interface.

##  Features

- Responsive design for desktop and mobile
- Modern UI with smooth scrolling and animations
- Interactive navigation bar
- About Me section
- Skills section
- Projects showcase
- Experience timeline
- Contact form
- Social media links
- Single-file implementation (HTML, CSS & JavaScript)

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts

##  Project Structure

```
Portfolio/
│── index.html
```

> This project is built entirely in a single HTML file with embedded CSS and JavaScript.

##  How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/likhitha-muddana-portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your browser

or

4. Open the folder in VS Code and use the **Live Server** extension.
